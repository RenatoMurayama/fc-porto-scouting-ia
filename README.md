# 🔵 FC Porto Data Office: Scouting Predictor ⚽

Este projeto utiliza **Machine Learning** para auxiliar a tomada de decisão no departamento de scouting do FC Porto, prevendo o valor de mercado de atletas com base em métricas de performance técnica.

## [Aceder ao App Online Aqui][https://fc-porto-scouting-ia-czmwdzjhih6tze8ey9kk6b.streamlit.app/]

### Objetivo do Projeto
Demonstrar a aplicação de análise preditiva no futebol moderno. O modelo identifica jogadores cujo valor de mercado está abaixo do valor previsto pela IA, sinalizando potenciais "oportunidades de negócio" para o clube.

### Tecnologias e Metodologia
- **Linguagem:** Python
- **Modelagem:** Random Forest Regressor (Scikit-Learn)
- **Interface:** Streamlit para visualização de dados interativa
- **Engenharia de Dados:** Tratamento de bases de dados do EA Sports FC 25
- **Visualização:** Gráficos de radar e dispersão com Plotly

### Funcionalidades
- **Análise Individual:** Perfil técnico detalhado comparado à média do mercado.
- **Exploração de Mercado:** Filtros inteligentes por OVR e Orçamento para encontrar talentos.
- **Predição de Valor:** Algoritmo que correlaciona atributos técnicos e ranking global.
